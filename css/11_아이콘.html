# CDN (Content Delivery Network)

  • 컨텐츠를 네트워크를 통해 제공하는 것
  • 컨텐츠 제공회사가 제공하는 여러가지 정적 자원들(글꼴,아이콘,이미지,JS,CSS)을 가져다 사용할 수 있다

# google icons (Material icon)

아이콘은 실제로는 글자이기 때문에 글자에 대한 스타일이 모두 적용된다.
※여러가지 타입(two-tone/sharp 등등)의 구글 아이콘을 적용할 때는 link태그의 url에 |(수직선)을 이용한다
ex)https://fonts.googleapis.com/icon?family=Material+Icons|Material+Icons+Two+Tone|Material+Icons+Sharp
