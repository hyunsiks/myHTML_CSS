# 페이지에 스타일시트를 적용하는 방법 3가지

1 inline CSS (줄 내부 CSS)
- style attribute를 이용해 스타일을 적용하는 방식
- css 적용 우선순위가 가장 높다
2 internal CSS (페이지 내부 CSS)
- style tag를 이용해 스타일을 적용하는 방식
- 더 나중에 나온 스타일시트의 우선순위가 높다
3 external CSS (페이지 외부 CSS)
- 외부의 .css 파일을 불러와 페이지에 적용하는 방식
- 외부의 .css 파일을 부를 때는 <link>태그를 이용한다.
- 우선 순위는 internal CSS와 같다
